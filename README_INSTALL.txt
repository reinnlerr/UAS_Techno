CARA MENJALANKAN FASILBOOK:
1. Pindahkan folder ini ke dalam folder C:\xampp\htdocs\
2. Buka XAMPP, jalankan Apache dan MySQL.
3. Buka browser, pergi ke http://localhost/phpmyadmin
4. Buat database baru dengan nama: fasilbook
5. Import file 'fasilbook.sql' yang ada di dalam folder 'sql'.
6. Buka web FasilBook di browser: http://localhost/FasilBook_Web_Final/

Catatan Library:
Karena ini adalah struktur yang meniru repo 'wartek', folder 'lib' sudah disediakan. 
Anda dapat menyalin file library asli dari folder 'wartek/lib' Anda ke folder 'lib' ini jika animasi WOW.js ingin diaktifkan secara offline.
